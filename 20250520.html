前回の復習: 解と係数の関係 📜

前回の授業では、2次方程式 \(ax^2+bx+c=0\) の2つの解を \(\alpha, \beta\) としたとき、以下の関係が成り立つことを学びました。

📜 ★★★
  • 解の和: \(\alpha + \beta = -\frac{b}{a}\)
  • 解の積: \(\alpha \beta = \frac{c}{a}\)

本日はこの関係を応用していくことと、新しいトピックとして整式の割り算について学習します。

2つの数から2次方程式を作る

解と係数の関係を利用すると、逆に2つの数からそれらを解として持つような2次方程式を作ることができます。

定理: 2解から2次方程式を構成する 📜

📜 ★★★

2つの数 \(\alpha, \beta\) を解とする2次方程式の1つは、次のように表すことができます。

\[ x^2 - (\alpha+\beta)x + \alpha\beta = 0 \]

これは、\((x-\alpha)(x-\beta)=0\) を展開した形と同じです。つまり、解の和と積が分かれば、2次方程式を構成できるということです。

例題1: 新しい解を持つ2次方程式の作成 ✍️ ★★☆ ★★★

2次方程式 \(x^2 - 2x + 5 = 0\) の2つの解を \(\alpha, \beta\) とするとき、\(\alpha^2, \beta^2\) を解とする2次方程式を1つ作りなさい。（教科書 例題4 改題）

解答・解説

まず、元の2次方程式 \(x^2 - 2x + 5 = 0\) について、解と係数の関係を用います。

\(\alpha + \beta = -(\frac{-2}{1}) = 2\)

\(\alpha\beta = \frac{5}{1} = 5\)

次に、新しい2次方程式の解は \(\alpha^2\) と \(\beta^2\) です。これらの和と積を求めます。

和: \(\alpha^2 + \beta^2\)

これは対称式なので、\(\alpha+\beta\) と \(\alpha\beta\) を用いて変形できます。

\(\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = (2)^2 - 2(5) = 4 - 10 = -6\)

積: \(\alpha^2 \beta^2\)

\(\alpha^2 \beta^2 = (\alpha\beta)^2 = (5)^2 = 25\)

したがって、\(\alpha^2, \beta^2\) を解とする2次方程式の1つは、解の和が \(-6\)、解の積が \(25\) なので、次のように表せます。

\(x^2 - (-6)x + 25 = 0\)

\(x^2 + 6x + 25 = 0\)

答え: \(x^2 + 6x + 25 = 0\)

複素数の範囲での因数分解

これまで、実数の範囲で因数分解できない2次式も、複素数の範囲まで考えを広げると因数分解できる場合があります。

考え方: 解を利用した因数分解 ⭐

⭐ ★★☆

2次方程式 \(ax^2+bx+c=0\) の2つの解が \(\alpha, \beta\) であるとき、2次式 \(ax^2+bx+c\) は次のように因数分解できます。

\[ ax^2+bx+c = a(x-\alpha)(x-\beta) \]

この関係を利用すると、解の公式を使って解を求めれば、どんな2次式でも（複素数の範囲を含めて）因数分解が可能になります。

例題2: 複素数の範囲での因数分解 ✍️ ★★☆ ★★★

2次式 \(x^2 - 2x + 4\) を複素数の範囲で因数分解しなさい。（教科書 例題の応用）

解答・解説

まず、2次方程式 \(x^2 - 2x + 4 = 0\) の解を解の公式を使って求めます。

\(a=1, b=-2, c=4\) なので、

\(x = \frac{-(-2) \pm \sqrt{(-2)^2 - 4 \cdot 1 \cdot 4}}{2 \cdot 1} = \frac{2 \pm \sqrt{4 - 16}}{2} = \frac{2 \pm \sqrt{-12}}{2}\)

\(\sqrt{-12} = \sqrt{12}i = \sqrt{4 \times 3}i = 2\sqrt{3}i\) なので、

\(x = \frac{2 \pm 2\sqrt{3}i}{2} = 1 \pm \sqrt{3}i\)

したがって、2つの解は \(\alpha = 1+\sqrt{3}i\) と \(\beta = 1-\sqrt{3}i\) です。

これらを用いて、\(a(x-\alpha)(x-\beta)\) の形（ここでは \(a=1\)）で因数分解します。

\(x^2 - 2x + 4 = \{x - (1+\sqrt{3}i)\}\{x - (1-\sqrt{3}i)\}\)

\(= (x - 1 - \sqrt{3}i)(x - 1 + \sqrt{3}i)\)

答え: \((x - 1 - \sqrt{3}i)(x - 1 + \sqrt{3}i)\)

⚠️

注意点: 解の公式の適用

★★☆ 解の公式を用いる際、特に \(b^2-4ac\) の部分が負になる場合（虚数解を持つ場合）は、平方根の計算と虚数単位 \(i\) の扱いに注意が必要です。

  • 例: \(\sqrt{-12}\) を計算する際に、\(2\sqrt{3}i\) と正しく変形できるようにしましょう。\(\sqrt{-12} = \sqrt{-1 \times 12} = i\sqrt{12} = i\sqrt{4 \times 3} = 2i\sqrt{3}\) (または \(2\sqrt{3}i\))。
  • 符号ミス: \(x-(1+\sqrt{3}i)\) を展開する際、\(x-1-\sqrt{3}i\) となるように、括弧を外すときの符号変化に気をつけましょう。

理解度チェッククイズ 1

  1. 2つの数 \(2+\sqrt{3}\) と \(2-\sqrt{3}\) を解とする2次方程式（\(x^2\)の係数は1）はどれですか？

    1. \(x^2 - 4x + 1 = 0\)
    2. \(x^2 + 4x + 1 = 0\)
    3. \(x^2 - 4x - 1 = 0\)
    4. \(x^2 + x - 4 = 0\)
    答えと解説

    答え: A. \(x^2 - 4x + 1 = 0\)

    解説: 解の和 \(\alpha+\beta = (2+\sqrt{3}) + (2-\sqrt{3}) = 4\)。解の積 \(\alpha\beta = (2+\sqrt{3})(2-\sqrt{3}) = 2^2 - (\sqrt{3})^2 = 4-3=1\)。よって、\(x^2 - (4)x + (1) = 0 \Rightarrow x^2 - 4x + 1 = 0\)。

  2. 2次式 \(x^2+9\) を複素数の範囲で因数分解した結果として正しいものはどれですか？

    1. \((x+3)(x-3)\)
    2. \((x+3i)(x-3i)\)
    3. \((x+9i)(x-9i)\)
    4. 因数分解できない
    答えと解説

    答え: B. \((x+3i)(x-3i)\)

    解説: \(x^2+9=0\) の解は \(x^2=-9 \Rightarrow x = \pm\sqrt{-9} = \pm 3i\)。よって、因数分解すると \((x-3i)(x-(-3i)) = (x-3i)(x+3i)\)。

整式の割り算

整数の割り算と同様に、整式（多項式）についても割り算を考えることができます。

整数の割り算との比較

例えば、整数 \(38\) を \(7\) で割ると、商は \(5\) で余りは \(3\) です。これは次の形で表せます。

\[ 38 = 7 \times 5 + 3 \]

（割られる数）＝（割る数）×（商）＋（余り）

整式の割り算も、これと同じような考え方で行います。

定義: 整式の割り算と余りの次数 📖

📖 ★★★

整式 \(A\) を整式 \(B\) (\(B \neq 0\)) で割ったときの商を \(Q\)、余りを \(R\) とすると、次の関係式が成り立ちます。

\[ A = BQ + R \]

このとき、(余り \(R\) の次数) ＜ (割る式 \(B\) の次数) となるか、または \(R=0\) (割り切れる場合) です。

⭐ ★★★

余りの次数が割る式の次数よりも必ず小さくなる、という点が非常に重要です。割り算の筆算を行う際、この条件を満たすまで計算を続けることになります。

例題3: 整式の割り算（筆算） ✍️ ★★☆ ★★★

整式 \(A = 2x^3 + 7x^2 + 5x + 9\) を整式 \(B = x+1\) で割りなさい。（教科書 例の応用）

解答・解説

整数の割り算と同様に、筆算で計算します。


        2x^2  + 5x   
      ________________
x+1 | 2x^3 + 7x^2 + 5x + 9
      -(2x^3 + 2x^2)
      _________________
             5x^2 + 5x
           -(5x^2 + 5x)
           ___________
                    0 + 9
                    (9)

計算手順の解説:

  1. まず、割られる式 \(2x^3 + 7x^2 + 5x + 9\) の最高次の項 (\(2x^3\)) と、割る式 \(x+1\) の最高次の項 (\(x\)) に注目します。\(2x^3 \div x = 2x^2\) なので、商の最初の項は \(2x^2\) です。
  2. \(2x^2\) と割る式 \((x+1)\) を掛け算します: \(2x^2(x+1) = 2x^3 + 2x^2\)。これを割られる式の下に書き、引き算します。 \((2x^3 + 7x^2) - (2x^3 + 2x^2) = 5x^2\)。
  3. 次に、割られる式の次の項 (\(5x\)) を下ろしてきて、\(5x^2+5x\) とします。
  4. 再び、\(5x^2+5x\) の最高次の項 (\(5x^2\)) と、割る式の最高次の項 (\(x\)) に注目します。\(5x^2 \div x = 5x\) なので、商の次の項は \(+5x\) です。
  5. \(5x\) と割る式 \((x+1)\) を掛け算します: \(5x(x+1) = 5x^2 + 5x\)。これを \(5x^2+5x\) の下に書き、引き算します。 \((5x^2 + 5x) - (5x^2 + 5x) = 0\)。
  6. 最後に、割られる式の次の項 (\(9\)) を下ろしてきます。残りは \(9\) です。
  7. 割る式 \(x+1\) は1次式、余り候補の \(9\) は0次式（定数項）なので、(余りの次数) ＜ (割る式の次数) を満たします。ここで割り算は終了です。

したがって、商は \(2x^2+5x\)、余りは \(9\) です。

関係式で表すと、\(2x^3 + 7x^2 + 5x + 9 = (x+1)(2x^2+5x) + 9\)

答え: 商 \(2x^2+5x\), 余り \(9\)

⚠️

注意点: 割り算の筆算における注意

★★★ 整式の割り算の筆算では、いくつかの注意点があります。

  • 次数の降べきの順に整理: 割られる式も割る式も、必ず次数の高い項から低い項の順（降べきの順）に整理してから筆算を始めます。
  • 抜けている次数はスペースを空けるか0で補う: 例えば、\(x^3 - x + 1\) を割る場合、\(x^2\) の項がありません。このような場合は、\(x^3 + 0x^2 - x + 1\) のように係数0で補うか、その部分のスペースを空けて筆算を行うと、同類項の計算ミスを防げます。
  • 引き算の符号ミス: 各ステップで引き算を行いますが、この際に符号のミスが非常によく起こります。引く式全体の符号が変わることを意識し、慎重に計算しましょう。例えば、\((A) - (B+C)\) は \((A) - B - C\) となります。
  • 計算の終了条件: 余りの次数が割る式の次数よりも小さくなったら計算終了です。

練習問題1: 整式の割り算 ✏️ ★★☆ ★★★

整式 \(A = x^3 - x + 1\) を整式 \(B = x-2\) で割りなさい。

解答・解説

割られる式 \(x^3 - x + 1\) には \(x^2\) の項がないため、\(x^3 + 0x^2 - x + 1\) として筆算を行います。


          x^2  + 2x   + 3
        _________________
x-2 | x^3 + 0x^2 -  x + 1
      -(x^3 - 2x^2)
      _________________
             2x^2 -  x
           -(2x^2 - 4x)
           ___________
                   3x + 1
                 -(3x - 6)
                 _________
                         7

したがって、商は \(x^2+2x+3\)、余りは \(7\) です。

関係式で表すと、\(x^3 - x + 1 = (x-2)(x^2+2x+3) + 7\)

答え: 商 \(x^2+2x+3\), 余り \(7\)

公式・定理の使い分けガイド

本日学習した内容の使い分けポイントです。

  • 解と係数の関係

    使う場面: 2次方程式の解そのものを求める必要がなく、解の和や積、またはそれらから導かれる対称式の値を求めたいとき。

    ポイント: \(ax^2+bx+c=0\) の解を \(\alpha, \beta\) とすると、\(\alpha+\beta = -b/a\), \(\alpha\beta = c/a\)。和の符号に注意。

  • 2解から2次方程式を構成する

    使う場面: 2つの数が与えられ、それらを解に持つ2次方程式を作りたいとき。または、既存の2次方程式の解から、新しい条件を満たす解を持つ2次方程式を作るとき。

    ポイント: 解を \(\alpha, \beta\) とすると、\(x^2 - (\text{和})x + (\text{積}) = 0\)。つまり、\(x^2 - (\alpha+\beta)x + \alpha\beta = 0\)。

  • 複素数の範囲での因数分解

    使う場面: 実数の範囲では因数分解できない2次式を、複素数まで範囲を広げて因数分解したいとき。

    ポイント: \(ax^2+bx+c=0\) の解を \(\alpha, \beta\) とすると、\(ax^2+bx+c = a(x-\alpha)(x-\beta)\)。解の公式で解を求め、この形に当てはめる。

  • 整式の割り算

    使う場面: ある整式を別の整式で割り、商と余りを求めたいとき。因数定理や剰余の定理の基礎となる重要な計算。

    ポイント: 筆算の手順を正確に実行する。次数を揃え、引き算の符号に注意。余りの次数が割る式の次数より小さくなるまで続ける。

理解度チェッククイズ 2

  1. 整式 \(2x^3 - 5x + 1\) を \(x-1\) で割ったときの余りはいくつですか？

    1. \(-2\)
    2. \(-1\)
    3. \(0\)
    4. \(1\)
    答えと解説

    答え: A. \(-2\)

    解説: 筆算で割り算を実行します。\((2x^3 + 0x^2 - 5x + 1) \div (x-1)\)。

    
        2x^2  + 2x   - 3
      _________________
x-1 | 2x^3 + 0x^2 - 5x + 1
      -(2x^3 - 2x^2)
      _________________
             2x^2 - 5x
           -(2x^2 - 2x)
           ___________
                  -3x + 1
                -(-3x + 3)
                __________
                        -2

    商は \(2x^2+2x-3\)、余りは \(-2\)。

  2. 整式 \(A\) を \(x+2\) で割ると、商が \(x-3\) で余りが \(4\) であった。整式 \(A\) を求めなさい。

    1. \(x^2 - x - 2\)
    2. \(x^2 - x - 6\)
    3. \(x^2 - 5x - 2\)
    4. \(x^2 + x - 2\)
    答えと解説

    答え: A. \(x^2 - x - 2\)

    解説: \(A = BQ + R\) の関係を使います。ここで \(B=x+2, Q=x-3, R=4\)。

    \(A = (x+2)(x-3) + 4 = (x^2 -3x + 2x - 6) + 4 = x^2 - x - 6 + 4 = x^2 - x - 2\)。

標準的な練習問題（追加生成）

  1. 2つの数 \(1+i\) と \(1-i\) を解とする2次方程式を1つ作りなさい。（ただし、\(x^2\) の係数は1とする） ★★☆ ★★☆
    解答・解説

    解の和: \((1+i) + (1-i) = 2\)

    解の積: \((1+i)(1-i) = 1^2 - i^2 = 1 - (-1) = 1+1 = 2\)

    よって、2次方程式は \(x^2 - (2)x + (2) = 0 \Rightarrow x^2 - 2x + 2 = 0\)

  2. 2次式 \(2x^2 + x + 1\) を複素数の範囲で因数分解しなさい。 ★★☆ ★★☆
    解答・解説

    方程式 \(2x^2+x+1=0\) の解を求めます。

    \(x = \frac{-1 \pm \sqrt{1^2 - 4 \cdot 2 \cdot 1}}{2 \cdot 2} = \frac{-1 \pm \sqrt{1-8}}{4} = \frac{-1 \pm \sqrt{-7}}{4} = \frac{-1 \pm \sqrt{7}i}{4}\)

    2つの解は \(\alpha = \frac{-1+\sqrt{7}i}{4}\), \(\beta = \frac{-1-\sqrt{7}i}{4}\)。

    因数分解すると、\(2(x - \frac{-1+\sqrt{7}i}{4})(x - \frac{-1-\sqrt{7}i}{4})\)

  3. 整式 \(x^3 - 2x^2 + 5\) を \(x+2\) で割ったときの商と余りを求めなさい。 ★★☆ ★★★
    解答・解説

    \(x^3 - 2x^2 + 0x + 5\) を \(x+2\) で割ります。

    
        x^2  - 4x   + 8
      _________________
x+2 | x^3 - 2x^2 + 0x + 5
      -(x^3 + 2x^2)
      _________________
            -4x^2 + 0x
          -(-4x^2 - 8x)
          ____________
                   8x + 5
                 -(8x + 16)
                 __________
                        -11

    商: \(x^2-4x+8\), 余り: \(-11\)

✏️

発展的な練習問題（追加生成）

  1. 2次方程式 \(x^2 - 3x + m = 0\) の1つの解が他の解の2倍であるとき、定数 \(m\) の値と2つの解を求めなさい。 ★★★ ★★☆
    解答・解説

    2つの解を \(\alpha, 2\alpha\) とおく。

    解と係数の関係より、

    和: \(\alpha + 2\alpha = -(\frac{-3}{1}) \Rightarrow 3\alpha = 3 \Rightarrow \alpha = 1\)

    積: \(\alpha \cdot 2\alpha = \frac{m}{1} \Rightarrow 2\alpha^2 = m\)

    \(\alpha=1\) を積の式に代入すると、\(m = 2(1)^2 = 2\)。

    2つの解は \(\alpha=1\) と \(2\alpha=2\)。

    答え: \(m=2\), 解は \(1, 2\)

  2. 整式 \(P(x) = x^4 - 3x^2 + ax + b\) が \(x^2-x-2\) で割り切れるとき、定数 \(a, b\) の値を求めなさい。 ★★★ ★★★
    解答・解説

    \(x^2-x-2 = (x-2)(x+1)\) と因数分解できる。

    \(P(x)\) が \((x-2)(x+1)\) で割り切れるということは、\(P(x)\) は \((x-2)\) でも \((x+1)\) でも割り切れる。

    因数定理より、\(P(2)=0\) かつ \(P(-1)=0\)。

    \(P(2) = 2^4 - 3(2^2) + a(2) + b = 16 - 12 + 2a + b = 4 + 2a + b = 0 \quad \cdots ①\)

    \(P(-1) = (-1)^4 - 3((-1)^2) + a(-1) + b = 1 - 3 - a + b = -2 - a + b = 0 \quad \cdots ②\)

    ①より \(2a+b = -4\)

    ②より \(-a+b = 2\)

    ① - ② より、\(3a = -6 \Rightarrow a = -2\)

    これを②に代入して、\(-(-2)+b=2 \Rightarrow 2+b=2 \Rightarrow b=0\)

    答え: \(a=-2, b=0\)
