前回の復習: 整式の割り算

前回の授業では、整式の割り算について学びました。整式 \(A\) を整式 \(B\) (\(B \neq 0\)) で割ったときの商を \(Q\)、余りを \(R\) とすると、

\[ A = BQ + R \]

という関係が成り立ち、(余り \(R\) の次数) ＜ (割る式 \(B\) の次数) または \(R=0\) となることを確認しました。筆算による割り算の練習も行いましたね。

✍️

復習例: 整式の割り算

例えば、\(A = 2x^3 - 5x + 7\) を \(B = x+1\) で割る場合、

\(A = (x+1)(2x^2 - 2x - 3) + 10\) のように表せます。

このとき、商 \(Q\) は \(2x^2 - 2x - 3\)、余り \(R\) は \(10\) です。

筆算の際には、割られる式にない次数の項は係数0で補う（例: \(2x^3 + 0x^2 - 5x + 7\)）と計算しやすくなります。

剰余の定理

整式の割り算において、余りだけを簡単に求めることができる便利な定理が「剰余の定理」です。

剰余の定理の定義 📜

📜 ★★★

整式 \(P(x)\) を一次式 \(x-a\) で割ったときの余りは、\(P(a)\) に等しい。

⭐ ★★★

解説:

整式 \(P(x)\) を \(x-a\) で割ったときの商を \(Q(x)\)、余りを \(R\)（これは定数）とすると、割り算の関係式から

\[ P(x) = (x-a)Q(x) + R \]

と書けます。この式は \(x\) についての恒等式なので、\(x\) にどんな値を代入しても成り立ちます。

ここで、\(x=a\) を代入すると、

\[ P(a) = (a-a)Q(a) + R = 0 \cdot Q(a) + R = R \]

したがって、余り \(R\) は \(P(a)\) に等しくなります。

この定理を使えば、実際に割り算の筆算をしなくても、特定の一次式で割ったときの余りを求めることができます。

⚠️

注意点: 割る式が \(x-a\) の形であること

★★☆ 剰余の定理が直接使えるのは、割る式が \(x-a\) という形の一次式の場合です。例えば、\(2x-1\) で割る場合は、一度 \(2(x-\frac{1}{2})\) のように変形して考える必要があります。

例題1: 剰余の定理を用いた余りの計算 ✍️ ★☆☆ ★★★

整式 \(P(x) = x^3 - 2x^2 + 3x + 5\) を \(x-1\) で割ったときの余りを求めなさい。

解答・解説

剰余の定理より、\(P(x)\) を \(x-1\) で割った余りは \(P(1)\) に等しい。

\(P(1) = (1)^3 - 2(1)^2 + 3(1) + 5 = 1 - 2 + 3 + 5 = 7\)

答え: 7

因数定理

剰余の定理を特別な場合として捉えると、因数定理が得られます。これは整式を因数分解する際に非常に役立ちます。

因数定理の定義 📜

📜 ★★★

整式 \(P(x)\) において、

  • \(P(a) = 0\) ならば、\(x-a\) は \(P(x)\) の因数である。
  • 逆に、\(x-a\) が \(P(x)\) の因数ならば、\(P(a) = 0\) である。
⭐ ★★★

解説:

剰余の定理 \(P(x) = (x-a)Q(x) + R\) において、もし余り \(R\) が \(0\) であれば、\(P(x) = (x-a)Q(x)\) となり、\(x-a\) は \(P(x)\) を割り切る因数であると言えます。

そして、剰余の定理から余り \(R = P(a)\) なので、\(P(a)=0\) であることが \(x-a\) が因数であるための条件となります。

この定理は、高次方程式の解を見つけたり、高次式を因数分解したりする際に、まず \(P(a)=0\) となる \(a\) を見つける手がかりとして使われます。

剰余の定理と因数定理の関係 ➕

➕

因数定理は、剰余の定理の特別な場合（余りが0の場合）と考えることができます。

  • 剰余の定理: \(P(x)\) を \(x-a\) で割った余りは \(P(a)\)。
  • 因数定理: \(P(x)\) が \(x-a\) で割り切れる（余り0） \(\iff P(a)=0\)。

例題2: 因数定理を用いた因数の判定 ✍️ ★★☆ ★★★

整式 \(P(x) = x^3 - 5x + 2\) が \(x-2\) を因数に持つかどうか調べなさい。

解答・解説

因数定理より、\(x-2\) が \(P(x)\) の因数であるかどうかは、\(P(2)\) の値を調べればよい。

\(P(2) = (2)^3 - 5(2) + 2 = 8 - 10 + 2 = 0\)

\(P(2)=0\) となったので、\(x-2\) は \(P(x)\) の因数である。

答え: \(x-2\) は因数に持つ。

練習問題1: 因数定理の利用 ✏️ ★★☆ ★★★

整式 \(P(x) = x^3 + x^2 - 5x - 2\) が \(x+2\) を因数に持つかどうか調べなさい。

解答・解説

因数定理より、\(x+2 = x-(-2)\) が \(P(x)\) の因数であるかどうかは、\(P(-2)\) の値を調べればよい。

\(P(-2) = (-2)^3 + (-2)^2 - 5(-2) - 2 = -8 + 4 + 10 - 2 = -10 + 14 = 4\)

\(P(-2)=4 \neq 0\) なので、\(x+2\) は \(P(x)\) の因数ではない。

答え: \(x+2\) は因数に持たない。

⚠️

間違いやすいポイント: P(a)=0の確認

★★☆ 因数定理を利用する際は、\(P(a)\) の計算を正確に行うことが重要です。特に負の数を代入する場合の符号ミスに注意しましょう。

  • 例: \(P(x) = x^3 - 5x + 2\) で \(x+2\) が因数か調べるために \(P(-2)\) を計算する際、\((-2)^3 = -8\) や \(-5(-2) = +10\) のような計算を誤らないように。
  • 対策: 代入計算は焦らず、一つ一つの項を丁寧に計算し、符号の変化に注意してください。\(P(a)=0\) になるかどうかで結論が真逆になるため、計算ミスは致命的です。

公式の使い分けガイド: 剰余の定理と因数定理

剰余の定理と因数定理は密接に関連していますが、目的によって使い分けます。

  • 剰余の定理 \(P(x) = (x-a)Q(x) + P(a)\)

    使う場面: 整式 \(P(x)\) を一次式 \(x-a\) で割ったときの余りだけを知りたい場合。

    ポイント: \(x=a\) を \(P(x)\) に代入するだけで余りが求まる。商を求める必要はない。

  • 因数定理 \(P(a)=0 \iff x-a \text{ は } P(x) \text{ の因数}\)

    使う場面: 整式 \(P(x)\) が \(x-a\) で割り切れるかどうかを判定したい場合。または、\(P(x)\) の因数を見つけたい場合（高次式の因数分解の手がかり）。

    ポイント: \(P(a)=0\) となる \(a\) を見つけることが鍵。\(a\) の候補は、一般に「(定数項の約数) / (最高次の項の係数の約数)」から探すことが多い（有理数解を持つ場合）。

理解度チェッククイズ 1

  1. 整式 \(P(x) = 2x^3 - x^2 + 5x - 3\) を \(x+1\) で割ったときの余りはいくつですか？

    1. \(-11\)
    2. \(-7\)
    3. \(3\)
    4. \(9\)
    答えと解説

    答え: A. \(-11\)

    解説: 剰余の定理より、余りは \(P(-1)\)。

    \(P(-1) = 2(-1)^3 - (-1)^2 + 5(-1) - 3 = 2(-1) - 1 - 5 - 3 = -2 - 1 - 5 - 3 = -11\)。

  2. 整式 \(P(x) = x^3 - 4x^2 + x + 6\) について、次のうち \(P(x)\) の因数であるものはどれですか？

    1. \(x-1\)
    2. \(x+1\)
    3. \(x-2\)
    4. \(x+2\)
    答えと解説

    答え: B. \(x+1\)

    解説: 因数定理を用いて確認します。

    A. \(P(1) = 1-4+1+6 = 4 \neq 0\)

    B. \(P(-1) = (-1)^3 - 4(-1)^2 + (-1) + 6 = -1 - 4 - 1 + 6 = 0\)。よって \(x-(-1)=x+1\) は因数。

    C. \(P(2) = 2^3 - 4(2^2) + 2 + 6 = 8 - 16 + 2 + 6 = 0\)。よって \(x-2\) も因数です。（問題の選択肢の意図としては、一つ選ぶ形式を想定しました。BとCの両方が正解となります。もし単一正解を求めるなら問題設定の調整が必要です。）

    D. \(P(-2) = (-2)^3 - 4(-2)^2 + (-2) + 6 = -8 - 16 - 2 + 6 = -20 \neq 0\)

    (追加解説) この問題の場合、Cも正解となります。複数の選択肢が正解となるケースもあります。 \(P(x) = (x+1)(x-2)(x-3)\) と因数分解できます。

本日のまとめ

  • 剰余の定理: 整式 \(P(x)\) を \(x-a\) で割った余りは \(P(a)\)。余りだけを求めたい時に有効。
  • 因数定理: \(P(a)=0 \iff (x-a)\) は \(P(x)\) の因数。因数分解の手がかりとなる。
  • これらの定理は、整式の割り算 \(A=BQ+R\) の関係式から導き出されます。
  • 特に、一次式で割る場合にこれらの定理は強力なツールとなります。

標準的な練習問題（追加生成）

  1. 整式 \(P(x) = x^4 - 3x^3 + 2x - 5\) を \(x-3\) で割ったときの余りを求めなさい。 ★★☆ ★★☆
    解答・解説

    剰余の定理より、余りは \(P(3)\)。

    \(P(3) = (3)^4 - 3(3)^3 + 2(3) - 5\)

    \(= 81 - 3(27) + 6 - 5\)

    \(= 81 - 81 + 6 - 5 = 1\)

    答え: 1

  2. 整式 \(P(x) = 2x^3 + ax^2 - x + 2\) が \(x+2\) で割り切れるとき、定数 \(a\) の値を求めなさい。 ★★☆ ★★★
    解答・解説

    因数定理より、\(P(x)\) が \(x+2\) で割り切れるならば \(P(-2)=0\)。

    \(P(-2) = 2(-2)^3 + a(-2)^2 - (-2) + 2\)

    \(= 2(-8) + a(4) + 2 + 2\)

    \(= -16 + 4a + 4 = 4a - 12\)

    \(P(-2)=0\) なので、\(4a-12=0 \Rightarrow 4a=12 \Rightarrow a=3\)。

    答え: \(a=3\)

✏️

発展的な練習問題（追加生成）

  1. 整式 \(P(x) = x^3 + ax^2 + bx - 6\) が \(x-1\) と \(x-2\) の両方で割り切れるとき、定数 \(a, b\) の値を求めなさい。 ★★★ ★★★
    解答・解説

    因数定理より、\(P(1)=0\) かつ \(P(2)=0\)。

    \(P(1) = 1^3 + a(1^2) + b(1) - 6 = 1 + a + b - 6 = a+b-5\)。

    よって、\(a+b-5=0 \Rightarrow a+b=5 \quad \cdots ①\)

    \(P(2) = 2^3 + a(2^2) + b(2) - 6 = 8 + 4a + 2b - 6 = 4a+2b+2\)。

    よって、\(4a+2b+2=0 \Rightarrow 2a+b=-1 \quad \cdots ②\)

    ② - ① より、\((2a+b) - (a+b) = -1 - 5\)

    \(a = -6\)

    これを①に代入して、\(-6+b=5 \Rightarrow b=11\)。

    答え: \(a=-6, b=11\)

  2. 整式 \(P(x)\) を \(x-1\) で割ると余りが \(5\)、\(x+3\) で割ると余りが \(-7\) となる。\(P(x)\) を \((x-1)(x+3)\) で割ったときの余りを求めなさい。 ★★★ ★★★
    解答・解説

    \(P(x)\) を \((x-1)(x+3)\)（これは2次式）で割ったときの商を \(Q(x)\)、余りを \(ax+b\) とおく。（割る式が2次式なので、余りは1次式以下になる）。

    すると、\(P(x) = (x-1)(x+3)Q(x) + ax+b\) と書ける。

    剰余の定理より、\(P(1)=5\) および \(P(-3)=-7\)。

    \(P(1) = (1-1)(1+3)Q(1) + a(1)+b = 0 \cdot Q(1) + a+b = a+b\)。

    よって、\(a+b=5 \quad \cdots ①\)

    \(P(-3) = (-3-1)(-3+3)Q(-3) + a(-3)+b = 0 \cdot Q(-3) -3a+b = -3a+b\)。

    よって、\(-3a+b=-7 \quad \cdots ②\)

    ① - ② より、\((a+b) - (-3a+b) = 5 - (-7)\)

    \(a+b+3a-b = 5+7\)

    \(4a = 12 \Rightarrow a=3\)。

    これを①に代入して、\(3+b=5 \Rightarrow b=2\)。

    したがって、余りは \(ax+b = 3x+2\)。

    答え: \(3x+2\)
